Modules Usuels de Manipulation des Matrices

25 modules algorithmiques pour manipuler des matrices. Type Mat pour les matrices, Tab pour les tableaux.

📌 Instructions générales
1Chaque module est autonome et traite une tâche spécifique.
2NL = lignes, NC = colonnes, N = taille d'une matrice carrée N×N.
3@M = passage par référence. Python : from numpy import array, accès M[i, j].
4Indices de 0 à NL-1. Pas de break ni return dans une boucle.

def RemplirDistinct(M, NL, NC):
    for i in range(NL):
        for j in range(NC):

Python
def distincts(M, L, C, NC):
    ok = True
    i = 0
    while i <= L and ok:
        j = 0
        while ((i < L and j < NC) or (i == L and j < C)) and ok:
            if M[i, j] == M[L, C]:
                ok = False
            else:
                j = j + 1
        i = i + 1
    return ok
🎯
Objectif
Remplir une matrice avec des entiers saisis, en garantissant que toutes les valeurs sont différentes.
💡
Idée générale
Pour chaque case, saisir une valeur puis vérifier avec distincts qu'elle n'existe pas déjà. Si doublon → redemander.
🔑
Mots-clés
i=0 et j=0Cas spécial : la première case est toujours acceptée.
distincts()Vérifie l'unicité en parcourant les cases déjà remplies.
📋
Étapes
  • 1Saisir M[i,j]
  • 2Si (i=0 et j=0) ou valeur distincte → accepter
  • 3Sinon → redemander jusqu'à obtenir une valeur valide
⚠️
Point d'attention
distincts ne vérifie que les cases déjà remplies, pas toute la matrice.

def InverserDiag1(M, N):
    for i in range(1, N):
        for j in range(i):
            aux = M[i, j]
            M[i, j] = M[j, i]
            M[j, i] = aux
🎯
Objectif
Échanger M[i,j] avec M[j,i] pour tout le triangle inférieur — c'est la transposition.
💡
Idée générale
On ne traite que le triangle inférieur (j < i) pour éviter de permuter deux fois.
🔑
Mots-clés
auxVariable temporaire indispensable pour l'échange.
i de 1 à N-1Commence à 1 car la ligne 0 n'a pas d'élément sous la diagonale.
📋
Étapes
  • 1Pour i de 1 à N-1, pour j de 0 à i-1
  • 2aux ← M[i,j]
  • 3M[i,j] ← M[j,i]
  • 4M[j,i] ← aux
⚠️
Point d'attention
Si on parcourait toute la matrice, chaque paire serait échangée deux fois — résultat identique à la matrice initiale.

SommeDiagonales(M, N) : entier
Calculer la somme des éléments des deux diagonales d'une matrice carrée N×N
▾
Algorithme
fonction SommeDiagonales (M : Mat, N : entier) : entier
début
    s ← 0
    pour i de 0 à N-1 faire
        s ← s + M[i, i]             // 1ère diagonale
        s ← s + M[i, N-i-1]   // 2ème diagonale
    fin pour
    si N mod 2 = 1 alors
        s ← s - M[N div 2, N div 2]  // centre compté 2 fois si N impair
    fin si
    retourner s
fin
Déclaration des objets locaux
ObjetType
sEntier
iEntier
ℹ️Déclaration numpy : M = array([[int()]*NC]*NL) — accès : M[i, j]
Python
def SommeDiagonales(M, N):
    s = 0
    for i in range(N):
        s = s + M[i, i]
        s = s + M[i, N - i - 1]
    if N % 2 == 1:
        s = s - M[N // 2, N // 2]
    return s
🎯
Objectif
Sommer les deux diagonales sans compter deux fois le centre si N est impair.
💡
Idée générale
En une boucle : M[i,i] (1ère diag) + M[i,N-i-1] (2ème diag). Si N impair, le centre appartient aux deux → le soustraire une fois.
🔑
Mots-clés
M[i, N-i-1]Position des éléments de la 2ème diagonale.
N mod 2 = 1Détecte si N est impair → élément central commun aux deux diagonales.
📋
Étapes
  • 1s ← 0
  • 2Pour i de 0 à N-1 : s ← s + M[i,i] + M[i,N-i-1]
  • 3Si N impair : s ← s - M[N div 2, N div 2]
  • 4Retourner s
⚠️
Point d'attention
Oublier la correction pour N impair → le centre est compté deux fois.

def MaximumMatrice(M, NL, NC):
    max = M[0, 0]
    for i in range(NL):
        for j in range(NC):

    for j in range(NC):
        mn = M[0, j]
        for i in range(1, NL):
            if M[i, j] < mn:
                mn = M[i, j]
        f.write(str(mn) + "\n")
    f.close()
🎯
Objectif
Pour chaque colonne, calculer son minimum et l'écrire dans le fichier — une valeur par ligne.
💡
Idée générale
Symétrique de SauvegarderMaxLigne mais on travaille sur les colonnes et on cherche le minimum.
🔑
Mots-clés
min ← M[0,j]Initialiser min avec le premier élément de chaque colonne.
boucle j externeOn traite colonne par colonne — j est l'indice fixe, i varie.
📋
Étapes
  • 1Ouvrir en écriture
  • 2Pour chaque colonne j : calculer min(M[0..NL-1,j])
  • 3Écrire min dans le fichier
  • 4Fermer
⚠️
Point d'attention
Le fichier contiendra exactement NC lignes, une par colonne de matrice.
